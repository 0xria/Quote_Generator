Quote Generator

This is a simple Quote Generator application that provides users with random inspirational quotes. The application fetches quotes from a predefined list and displays them to the user.

Features:
=> Instant quote generation...
=> User-friendly interface...
=> Responsive design for various devices...
=> Has the clipboard, new quote and twitter share buttons...

Tech Stack:
=> React Js
=> Typescript
=> Vite
=> CSS

Getting Started:
=> git clone https://github.com/0xria/Quote_Generator.git
=> cd Quote
=> npm install
=> npm run dev
